1. 输入inputn    
2. t = input // 226    
3. 计算a = input*input - t*input    
4. 计算(1^3 + 2^3 + 3^3 + ... + a^3) % n = b，可以化成: (a*(a+1)/2)^2 % n = b    
5. 对b进行rsa运算，各种参数在rsa函数中可见
